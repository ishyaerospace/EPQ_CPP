# structs in C++

C++ structures are a way to group several related variables into one place.

each variable in the struct is known as a member of the structure.
a struct can contain many different data types.


# struct syntax
`struct {             // Structure declaration`
`  int myNum;         // Member (int variable)`
`  string myString;   // Member (string variable)`
`} myStructure, myStructure1, myStructure2;// Structure variable, you can assign a structure to multiple variables`

# accessing structs

to access a member of a struct you need to use dot notation.
for example:
`myStruct.Value1 = 1;`
